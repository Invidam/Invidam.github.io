## 5. 데이터 타입 - 문자열과 숫자

+ 콘솔창에 1+1를 입력하면 2가 뜬다.
	+ + : 이항 연산자, 산술 연산자

+ 문자열은 "~~" or '~~'로 나타낸다.
	+ str.length : 문자열 길이 출력
		+ length : properties.
	+ str1.indexOf(str2) : str1에서, str2의 시작위치를 출력

+ 문자열과 숫자는 다른 데이터 타입이다.
