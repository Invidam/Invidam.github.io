## 문서의 구조와 슈퍼스타들
+ ```<title></title>``` : 이름을 명시 + 검색엔진의 잘 뜸
+ utf-8 : 한글 출력가능하게 하는 형식. 한글 출력위해서는 웹브라우저에도 적용해야함.
	+ ```<meta charset="utf-8">``` : 한글 출력위한 태그
+ ```<body></body>``` : 본문을 묶는 태그
+ ```<head></head>``` : 머리말을 묶는 태그
+ ```<html></html>``` : 모두를 묶는 태그
+ ```<!doctype html>```: html로 작성했음을 나타내는 관용적 표현 태그
